QUICK START COMMANDS
====================

1. Install Dependencies:
   pip install -r requirements.txt

2. Generate Sample Data:
   python test_sample_data.py

3. Launch Dashboard:
   streamlit run app.py

4. Or use automation:
   python setup.py

5. Windows users:
   Double-click run_dashboard.bat
